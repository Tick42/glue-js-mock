This is a simple Glue42 enabled application that uses Interop and Channels. It has the following functionalities:
    
1. Shows the data from invoking the "set-text" Interop method here:
2. Shows data from invoking the external "get-text" Interop method by clicking the button.
3. Shows the data in the currently selected color Channel:
4. Updates the data in the current color Channel (adds a `time` property to the Channel context with the current time as a value)
